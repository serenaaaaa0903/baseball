# **AIS (AI Scouting) - Project Blueprint**

## **1. Project Overview**
AIS는 야구 선수의 투구 및 타격 폼을 AI로 분석하여 전문적인 스카우팅 리포트를 제공하는 최첨단 플랫폼입니다. 2026년형 **High-End AI Dashboard** UI를 통해 전문가 수준의 분석 경험을 제공합니다.

## **2. Design & Aesthetics (Professional Overhaul)**
*   **Color Palette:** Obsidian Black (#020408), Deep Slate (#0c111d), Electric Blue (#3b82f6), and Emerald Green for success states.
*   **UI System:** Glassmorphism 2.0 with backdrop-blur (30px) and 1px hairline borders for a premium feel.
*   **Layout:** Logic-driven Bento Dashboard optimized for multi-screen and mobile.
*   **Typography:** 'Inter' for data precision and 'Orbitron' for kinetic branding.

## **3. Key Features**
*   **Pro-Level AI Analysis:** Non-duplicate real-time feedback from a virtual 'Expert Director'.
*   **Tiered Intelligence:**
    *   **Basic:** Core feedback & Video stream.
    *   **Pro:** Kinetic chain stability, Torque metrics, PDF Export, and Personalized Nutrition.
    *   **Super Pro:** Daily performance trends (Chart.js), shared intelligence portal, and detailed bio-recipes.
*   **Viral Growth System:** Friend invite modal with 1-time Free Pro Pass activation.

## **4. Current Implementation Plan (UI Overhaul)**
1.  **Dashboard Refactoring:** Transition from a simple grid to a professional-grade analytics dashboard.
2.  **Advanced Metrics UI:** Implement high-precision gauges and live biometrics displays.
3.  **Refined Component System:** Professional-grade buttons, input fields, and modals with smooth micro-interactions.
4.  **Consistency Check:** Ensure all elements (Chat, Ads, Logs) follow the new obsidian-slate design language.
